たまごクラス

目的・目標
  • 「英語ってたのしい！」というワクワク感を感じる
身に付く力
  • Yes[No]の返答と10までの数字がわかり、アルファベットの文字の形や発音を認識できる
  • 人と出会った時の最初と最後のあいさつ、簡単なやりとりがわかる

    • 表現例

    • Hi[Hello]. / Bye[See you]. / Good morning[night].
    • How are you? / Good[Fine]. / Thank you[thanks].
    • My name is[I am] ~. / I’m ~（歳）.
  • 自分の国の名前や食べ物、動物、色、体の部位などの簡単な単語がわかる

ひよこクラス

目的・目標
  • 「英語がわかる！言える！」のワクワク感を体感する
身に付く力
  • Yes[No]の返答と10までの数字が言え、アルファベットを正しく発音できる
  • 人と出会った時の最初と最後のあいさつ、簡単なやりとりが言える

    • 表現例

    • Hi[Hello]. / Bye[See you]. / Good morning[night].
    • How are you? / Good[Fine]. / Thank you[thanks].
    • My name is[I am] ~. / I’m ~（歳）.
  • 自分の国の名前や食べ物、動物、色、体の部位などの簡単な単語が言える
